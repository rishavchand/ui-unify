## ğŸ“¥ Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/PankajKumardev/ui-unify.git
   cd UI-Unify
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Set up environment variables:
   - Copy `.env.example` to `.env`.
   - Add required keys.
4. Start the development server:
   ```bash
   npm run dev
   ```
